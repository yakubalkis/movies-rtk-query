# Movies App

## Description

This project is a personal application built with React.js, TypeScript and RTK(Redux Toolkit), using Vite.js as the build tool. It integrates various libraries and tools for state management, styling, routing, data fetching, and more. <br>

It is a movie application where users can get information about movies, search and filter movies. <br>

### Used techniques
• Get debounced input value from textfield for api efficiency <br>
• Server-side pagination <br>
• RTK query for api calls <br>
• Fresh api calls by filtering <br>
